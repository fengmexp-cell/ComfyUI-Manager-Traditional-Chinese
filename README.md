ComfyUI-Manager 介面翻譯安裝教學
本倉庫提供 ComfyUI-Manager 的繁中/簡中翻譯檔案。
安裝步驟：
取得檔案：
根據需求下載 manager_zh_tw.js 或 manager_zh_cn.js。
存放到指定目錄：
將檔案放入您 Ubuntu 系統中的路徑：
ComfyUI/custom_nodes/ComfyUI-Manager/js/
直接使用

生效：
按下 Ctrl + F5 強制重新整理瀏覽器即可看到中文介面。
