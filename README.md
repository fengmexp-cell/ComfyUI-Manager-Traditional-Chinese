Installation:

Download file:
manager_zh_tw.js or manager_zh_cn.js

Place into:
ComfyUI/custom_nodes/ComfyUI-Manager/js/

Refresh browser:
Press Ctrl + F5

After installation:
Refresh the page (Ctrl + F5) to apply the Chinese interface.


本倉庫提供 ComfyUI-Manager 的繁體中文與簡體中文介面翻譯插件，用於改善 UI 使用體驗。
